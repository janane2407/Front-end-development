  VISION

   In a study, we have compared various machine learning algorithms and found the best algorithm used to classify tweets related to the disaster. Consolidating this data can offer assistance to post-disaster administration groups to make way better and more informed decisions. Studies have utilized Twitter information to extend awareness about the situations. Twitter has moreover made the openings for first responders to know the critical information and work effectively for the impacted communities. This twitter information can too be utilized to assess post-conditions. Disaster classification could be an issue where the main focus is to predict whether a given tweet is about a real disaster or not. The disaster tweet classification accuracy is compared with Linear Support Vector Classifier(SVC), Support Vector Machine(SVM), Random Forest(RF), Decision Tree, K-nearest neighbors(KNN), Logistic Regression(LR), and Naive Bayesian(NB) models. LR is found to be the optimum model for this classification. Further, the above-trained model can be used for disaster prediction on new Twitter data.
